# AntzGameOS

基于[AntzOS](https://github.com/CasterWx/AntzOS)的RPG冒险游戏。(吃饭的时候想到的坑，更新速度可能很慢)

![1](screen/VID_20191019_173937.gif)

引入NPC，新手村指导员Linus

![2](screen/linus.png)

##### 如何使用

下载仓库内tools包中的USB Image Tool启动盘制作工具与antz.img镜像文件。
准备一个空闲的U盘。
